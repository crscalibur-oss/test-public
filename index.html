常见停滞问题

  • 更新方向不适应市场需求（商业配比过高、产业方向不符等）
  • 土地移交率过高，导致项目经济不可行
  • 商办超配难以去化，影响项目现金流
  • 保障性住房配建要求增加开发成本

政策支持

更新方向优化

  • • 商改居（战略预留区外）
  • • 新型产业改普通工业

容积测算新规

  • • 基础容积按《深标》基准取值
  • • 转移容积不再增加30%奖励
  • • 公共设施计入奖励容积（1倍）

土地移交率与保障房

  • • 原则上≤35%（超35%需专项论证）
  • • 可不配建保障性住房

盘活路径

① 申请更新方向调整 ② 重新编制规划 ③ 按新规则测算容积 ④ 合理核定土地移交率 ⑤ 优化商业配比 ⑥ 报规划审批

关键操作要点

▶ 更新方向调整

商改居需位于战略预留区外，现状为旧商业区的商改商项目除外；新型产业改普通工业鼓励调整

▶ 容积测算新规

基础容积按《深标》密度分区基准取值；转移容积不再增加30%奖励；公共设施计入奖励容积

▶ 土地移交率

原则上≤35%，超35%需专项论证；实际移交面积≥基准移交面积

适用政策：深规划资源规〔2025〕5号第（八）（九）条；深建规〔2026〕3号第（三）（四）条

常见停滞问题

  • 原实施主体资金困难，无法继续推进
  • 商业面积难以去化，项目经济测算不成立
  • 经济关系复杂，多方无法达成一致

政策支持

规划优化调整（无需调整计划）

  • • 减少商业面积（非强制修改）
  • • 新型产业改普通工业
  • • 商业改配租型保障房（强制修改）

"清调供"机制

  • • 区政府厘清经济/法律关系
  • • 清退原实施方
  • • 引入新实施方或政府主导

金融支持

  • • 信贷服务、并购贷款
  • • 资产重组、破产重整

盘活路径

路径A（可协商）

规划优化调整 → 按相应程序修改规划 → 继续推进

路径B（无法协商）

启动"清调供"机制 → 厘清关系 → 清退主体 → 引入新主体

关键操作要点

▶ 规划优化类型

减少商业：非商业面积不变，可直接在用地批准阶段确定；产业改工业按非强制内容修改程序；商业改保障房需位于战略预留区外或轨道500米内

▶ "清调供"机制

区政府厘清经济关系及法律关系 → 清退原实施方 → 市规划资源部门组织编制规划 → 引入新实施方或政府主导实施

▶ 配套支持

金融机构提供信贷、并购贷款；金融资产管理公司资产重组、破产重整

适用政策：深规划资源规〔2025〕5号第（十）（十一）（十二）条

常见停滞问题

  • 实施主体无力继续开发建设
  • 资金链断裂，无法启动施工
  • 项目长期闲置，土地资源浪费

政策支持

"收储供"机制

  • • 区政府厘清经济/法律关系
  • • 土地收购纳入土地储备
  • • 重新进行土地供应

存量商办转换

  • • 不改变出让合同、不变更使用权人
  • • 商业/办公/旅馆可相互临时转换
  • • 鼓励转为公共服务设施、保障房

其他支持

  • • 多元化安置方式
  • • 补交超期地价

盘活路径

① 厘清经济/法律关系 ② 土地收购纳入储备 ③ 重新土地供应 ④ 引入新主体实施

关键操作要点

▶ "收储供"机制

区政府厘清经济关系及法律关系 → 相关部门按职能分工进行土地收购 → 纳入土地储备后重新供应

▶ 存量商办转换

不改变土地出让合同、不变更使用权人，商业/办公/旅馆可相互临时转换，鼓励转为公共服务设施、保障房等

▶ 其他支持

已拆除房屋政府安置采用多元化方式；土地使用权期限届满项目补交超期地价

推荐路径：更新方向调整

阶段一：单元立项后、规划审批前

可采取的政策支持：

  • • 商改居（战略预留区外）
  • • 新型产业改普通工业

盘活步骤：

申请更新方向调整 重新编制规划 按新规则测算容积 报规划审批

推荐路径：规划指标优化

阶段一：单元立项后、规划审批前

可采取的政策支持：

  • • 土地移交率原则上≤35%
  • • 可不配建保障性住房
  • • 按新规则测算容积

盘活步骤：

合理核定土地移交率 优化商业配比 重新编制规划 报规划审批

推荐路径：规划优化调整

阶段二：规划已批、合同未签

可采取的政策支持：

  • • 减少商业面积（非强制修改程序）
  • • 新型产业改普通工业
  • • 商业改配租型保障房（强制修改程序）

盘活步骤：

规划优化调整 按相应程序修改 继续推进

推荐路径："清调供"机制

阶段二：规划已批、合同未签

可采取的政策支持：

  • • 区政府厘清经济/法律关系
  • • 清退原实施方
  • • 金融支持（信贷、并购贷款、资产重组）

盘活步骤：

启动清调供 厘清关系 清退主体 引入新主体

推荐路径："收储供"机制

阶段三：合同已签、尚未开工

可采取的政策支持：

  • • 区政府厘清经济/法律关系
  • • 土地收购纳入土地储备
  • • 重新进行土地供应

盘活步骤：

厘清关系 土地收购 纳入储备 重新供应

推荐路径：存量商办转换 + 金融支持

阶段三：合同已签、尚未开工

可采取的政策支持：

  • • 存量商办用途临时转换（不改变出让合同）
  • • 转为公共服务设施、保障房等
  • • 金融支持（信贷、并购贷款）

盘活步骤：

申请用途转换 转为公共服务 获取金融支持
